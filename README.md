# 🌸 Flower Classification Using Convolutional Neural Networks (CNN)

## Project Overview
This project demonstrates image classification using a Convolutional Neural Network (CNN) to identify different flower species from images. The model was developed as part of my AI/ML learning journey and explores image preprocessing, model training, and prediction techniques.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- OpenCV
- Scikit-Image

## Key Features
- Image preprocessing and normalization
- CNN-based flower classification
- Model training and evaluation
- Prediction on unseen flower images

## Dataset
The project uses the 17 Flowers Dataset for image classification experiments.

## Learning Outcomes
- Understanding CNN architecture
- Image preprocessing techniques
- Deep learning model training
- Performance evaluation and prediction workflows

- `Flower_Classification.ipynb` – Main notebook
